# Step 3: Physiological Signal Driver Identification

## PURPOSE
Identifies the specific embedding dimensions that most strongly differentiate between language conditions (Spanish vs. English), serving as candidate "physiological drivers" for linguistic respiratory baseline.

This implements the core search described in the paper: looking for dimensions where foundation model representations capture stable, task-invariant physiological structure rather than linguistic content or recording artifacts.

**Note**: Edit this pipeline to point to other data sets to get LOSO for each etc.

## PAPER REFERENCE
- **Section 5**: A Stability-First Auditing Framework

Specifically addresses the paper's core question: *"How can we determine whether a representation extracted by a foundation audio model constitutes a valid physiological signal before it is used for screening?"*

## KEY SCIENTIFIC HYPOTHESES BEING TESTED

### 1. THE "CLIFF" HYPOTHESIS
If a small subset of dimensions (e.g., top 10) shows dramatically higher weights than the rest, this suggests the model has identified specific, concentrated physiological parameters rather than distributed noise.

### 2. THE "INTEGRATED PROCESSING" HYPOTHESIS
If physiological drivers are spread across the embedding space, it proves the model integrates information from the entire 2-second window (the "Hardware" - full respiratory cycle) rather than focusing on specific phonemes or vowel sounds (the "Software" - linguistic content).

### 3. THE "LINGUISTIC INVARIANCE" TEST
By training to distinguish Spanish vs. English, then examining which dimensions the classifier relies on, we can identify:
- **Dimensions that capture LANGUAGE-SPECIFIC artifacts** (should be rejected)
- **Dimensions that capture LANGUAGE-INVARIANT physiology** (candidates for retention)

## METHODOLOGICAL INNOVATION
This approach inverts traditional feature selection:

| Traditional Approach | This Approach |
|---------------------|---------------|
| "Find features that best predict disease" | "Find features that CAN predict language → then REJECT them" |

**Rationale**: Any dimension that strongly signals language difference is likely capturing linguistic/recording confounds rather than pure physiology.

## OUTPUT INTERPRETATION
Two possible outcomes with distinct scientific meanings:

### 1. "CLIFF" PATTERN (Concentrated Drivers)
- **Pattern**: Top 10 dimensions have weights >> remaining 502 dimensions
- **Implication**: Model has identified specific, interpretable physiological parameters
- **Paper Relevance**: These dimensions are strong candidates for stability auditing

### 2. "DISTRIBUTED" PATTERN (Spread Drivers)
- **Pattern**: Weights gradually decline without sharp drop-off
- **Implication**: Model integrates holistic window information
- **Paper Relevance**: Physiological signal may be encoded in distributed patterns rather than individual dimensions
